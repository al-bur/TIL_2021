## 작성법 
- 의미없는 내용으로 의미없는 TIL 작성은 하지 않도록 한다
- TIL에 작성하는 내용은 최소한 다른 사람에게 설명할 수 있도록 하자
- ex) [210118 / 제목] + (link주소)

<br>

## commit convention
- CREATE: 새로운 파일 작성 시
- UPDATE: 내용을 추가하거나
- FIX: 내용을 수정했을 시
- DOCS: 최상단의 README.md 파일(repo 설명하는 파일)에 관한 작업 진행 시
